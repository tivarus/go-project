# Bank go

## Технологии и стек 
|Компонент | Технологии|
|-----------------|---------------------------|
|Язык | Go 1.23+.|
|Маршрутизация | mux.|
|Работа с БД | PostgreSQL + lib/pq.|
|Аутентификация | golang-jwt.|
|Логирование | logrus.|
|Дополнительно | gomail.v2 (отправка email), beevik/etree (парсинг XML).|
|База данных | PostgreSQL 17 с расширением pgcrypto.|
